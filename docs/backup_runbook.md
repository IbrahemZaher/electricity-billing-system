# دليل استعادة النسخ الاحتياطية لنظام فواتير مولدة الريان

## المتطلبات
- خادم PostgreSQL مثبت (نفس الإصدار المستخدم في الإنتاج)
- أدوات عميل PostgreSQL (`pg_basebackup`, `pg_restore`, `psql`)
- مفاتيح GPG إذا كانت النسخ مشفرة
- صلاحيات الوصول إلى مجلدات النسخ الاحتياطي

## استعادة كاملة (Full Restore)

### الخطوات
1. أوقف تطبيق النظام لمنع الاتصالات بقاعدة البيانات.
2. حدد آخر نسخة احتياطية كاملة من مجلد `backups/` (الملف بامتداد `.backup` أو `.backup.gpg`).
3. إذا كانت النسخة مشفرة، قم بفك تشفيرها:
   ```bash
   gpg --decrypt backup_file.backup.gpg > backup_file.backup
